# YouTube Video Downloader & Summarizer

一個基於 Flask 和 React 的 YouTube 視頻下載和 AI 摘要生成工具。

## 功能特點

- 支持 YouTube 視頻下載（MP4 格式，QuickTime 兼容）
- 自動獲取視頻字幕
- 使用 GPT-3.5 生成視頻內容摘要
- 實時顯示下載進度
- 支持視頻播放預覽

## 技術棧

### 後端
- Flask
- yt-dlp
- OpenAI API
- FFmpeg

### 前端
- React
- Axios
- CSS3

## 環境要求

- Docker
- Docker Compose
- OpenAI API Key
